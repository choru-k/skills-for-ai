# Lifecycle Actions

Supported actions:
1. list personal projects (active + archived)
2. show active plans for a project
3. create/update notes in `notes/*.md`
4. update `repos.md`
5. archive project to `4-archive/personal/`
6. reopen archived project back to `1-projects/personal/`

Always preserve existing notes and progress.
