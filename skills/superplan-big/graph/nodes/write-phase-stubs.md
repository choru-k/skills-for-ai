# Write Phase Stubs

Create `phase-*.md` files at ticket root only.

Each stub must include:
- phase goal
- scope in/out
- dependencies
- exit criteria
- expand instruction: run `/superplan medium`

Do not create phase directories or item/task files in big tier.
