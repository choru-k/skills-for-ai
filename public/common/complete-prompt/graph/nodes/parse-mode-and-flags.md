# Parse Mode and Flags

Supported modes:
- `full` (default)
- `brief`
- `debug`
- `architect`
- `diff`
- `general`
- `research`
- `career`
- `learning`

Supported modifier:
- `--refs`

Output:
- selected mode
- refs mode enabled/disabled
