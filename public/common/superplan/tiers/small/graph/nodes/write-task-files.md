# Write Task Files

Before generation, read `tiers/small/references/tdd-workflow.md`.

Create `task-*.md` files where each task includes:
1. Overview
2. Scope
3. Write test
4. Verify fail
5. Implement
6. Verify pass
7. Commit
8. Error protocol
9. Dependencies
10. Exit criteria

Keep one concern per task file.
