# Load Template

Map mode to template file:
- `full` -> `templates/full.xml`
- `brief` -> `templates/brief.xml`
- `debug` -> `templates/debug.xml`
- `architect` -> `templates/architect.xml`
- `diff` -> `templates/diff.xml`
- `general` -> `templates/general.xml`
- `research` -> `templates/research.xml`
- `career` -> `templates/career.xml`
- `learning` -> `templates/learning.xml`

Always read template from disk; do not hallucinate schema.
